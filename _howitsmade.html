Jak to działa¶

Dodawanie obrazków¶

Gdy dodajemy obraz do bazy danych / sklepu (np korzystając z image_add) mają miejsce następujące wydarzenia:

  1. Obraz jest odczytywany z lokalnej ścieżki w systemie plików
  2. Program pobiera jego mimietype, a na tej podstawie pobiera jego filetypeid korzystając z tabeli filetype
  3. Program identyfikuje id jego rozszerzenia na podstawie tabeli fileextensionid
  4. Dodawany jest rekord do bazy danych identyfikujący plik, korzystając z wcześniej uzyskanych danych, uzyskujemy id zdjęcia
  5. Obraz jest kopiowany do design/_gallery/orginal/<id>.<ext>
  6. Dla każdego z rozmiarów (50, 100, 200, 300) program generuje miniaturkę i umieszcza ją w katalogu design/_gallery/_<size>_<size>_<shopid>/<id>.<ext>
